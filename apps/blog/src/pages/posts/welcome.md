# Welcome to Minions Bundle: Finance

We're excited to introduce **Minions Bundle: Finance** — Financial tracking bundle — budgets, expenses, invoices.

## What's Included

- **TypeScript SDK**: `npm install @mxn2020/minions-bundles-finance-core`
- **Python SDK**: `pip install minions-bundles-finance`
- **CLI**: `npm install -g @mxn2020/minions-bundles-finance-cli`

Check out the [documentation](https://docs.minions.wtf) for a full guide.
