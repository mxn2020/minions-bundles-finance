# Minions Bundle: Finance

Financial tracking bundle — budgets, expenses, invoices

## Overview

This project is part of the [Minions ecosystem](https://github.com/mxn2020/minions) and provides:

- **TypeScript SDK** (`@mxn2020/minions-bundles-finance-sdk`) — Core library for Node.js
- **Python SDK** (`minions-bundles-finance`) — Python bindings
- **CLI Tool** (`@mxn2020/minions-bundles-finance-cli`) — Command-line interface
- **Web App** — Interactive playground at [app.minions.wtf](https://app.minions.wtf)
